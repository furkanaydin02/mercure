# symfony6-php8 mercure

Bu proje, gerçek zamanlı bildirimler eklemek için kullanılan Mercure bileşenini içerir. Aşağıda, projeyi daha hızlı bir şekilde indirip kurmanız için talimatları bulabilirsiniz.

## Installation

```bash
make setup
```

```bash
make prepare
```
